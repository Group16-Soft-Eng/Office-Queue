# Folder Structure

```
src/
├── components/   # Reusable UI components (e.g., buttons, modals, ticket displays)
├── pages/        # Main page components (e.g., Home, Customer)
├── api/          # API request handlers (e.g., ticketApi.ts, counterApi.ts)
├── hooks/        # Custom React hooks (e.g.,useTicket.ts)
├── utils/        # Utility functions and helpers
├── assets/       # Static files (images, icons, SVGs)
├── styles/       # Global or modular CSS/SCSS files
├── App.tsx       # Main application component
└── main.tsx      # React application entry point
```
